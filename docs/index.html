Features

필요한 기능은 모두

복잡한 로그 분석도, 간단한 실시간 모니터링도 Logr 하나로.

📡

실시간 Follow 모드

파일이 업데이트될 때마다 자동으로 마지막 줄로 스크롤. F 키 하나로 켜고 끄기.

📂

멀티탭

여러 로그 파일을 탭으로 동시에 열고 관리. 각 탭마다 독립적인 검색·필터 상태.

🔍

검색 & 정규식

키워드 검색부터 정규식, 대소문자 구분까지. 이전/다음 매칭 줄로 빠르게 이동.

🎛️

레벨 필터

ERROR, WARN, INFO, DEBUG 레벨 자동 감지. 포함/제외 텍스트 필터로 원하는 줄만 표시.

🌐

SSH 원격 파일

서버에 직접 접속 없이 SSH 연결로 원격 로그 파일을 로컬처럼 모니터링.

✨

하이라이트 규칙

키워드별 색상 지정. 중요한 패턴을 한눈에 구분할 수 있도록 강조 표시.

🔤

인코딩 자동 감지

UTF-8, EUC-KR, CP949, UTF-16 자동 감지 및 수동 변경. 레거시 한글 로그도 정상 표시.

⭐

즐겨찾기

자주 보는 로그 파일을 즐겨찾기에 추가. 그룹, 별칭, 마지막 위치까지 기억.

📋

내보내기

현재 표시된 로그를 .txt 또는 .csv 파일로 저장. 팀원과 로그 공유도 간편하게.

FAQ

자주 묻는 질문

무료인가요?

네, 완전 무료 오픈소스입니다. MIT 라이선스로 소스 코드도 공개되어 있습니다.

macOS에서 "개발자를 확인할 수 없습니다" 오류가 떠요.

Apple 공증 인증서가 없어서 발생하는 경고입니다. 앱 파일을 우클릭 → "열기" → "열기" 버튼을 클릭하면 실행됩니다.
또는 터미널에서: xattr -cr /Applications/Logr.app

어떤 로그 파일 형식을 지원하나요?

일반 텍스트(.log, .txt)면 모두 지원합니다. 인코딩은 UTF-8, EUC-KR, CP949, UTF-16을 자동 감지하거나 수동으로 지정할 수 있습니다.

대용량 파일도 버벼나요?

가상 스크롤 기술로 수백만 줄 파일도 빠르게 처리합니다. 메모리에는 최대 20만 줄만 유지합니다.

SSH 원격 파일 연결은 어떻게 하나요?

사이드바의 SSH 섹션에서 "+" 버튼으로 서버 정보를 입력하면 됩니다. 비밀번호·SSH 키·SSH 에이전트 방식을 모두 지원합니다.
